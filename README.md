# Roxxon-News

Roxxon-News is a city news feed website inspired by the social app in Marvel’s Spider-Man: Miles Morales.

The website allows users to:
- Post comments about events in the city
- Reply to other comments
- See the current weather
- View the city feed in real time

## Features

- Live city comment feed
- Time-stamped posts
- Weather display
- City banner
- Spider-Man inspired interface

## Built With

- HTML
- CSS
- JavaScript

## Future Updates

- User accounts
- Reply threads
- Spider-Man sightings
- Crime alerts
- Real-time news updates

## Project Goal

The goal of Roxxon-News is to recreate a city social network similar to the one seen in the game Marvel’s Spider-Man: Miles Morales.
